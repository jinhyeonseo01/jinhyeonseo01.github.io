# Glossary

- **Hub**: The localized dashboard (`/{locale}/`) that routes users to destinations.
- **Registry**: JSON data files that define links and localized metadata.
- **Locale**: Supported language code (`ko`, `en`, `ja`, `zh-cn`).
- **Stable link**: Redirect endpoint `/go/{slug}/` used as a durable handoff URL.
- **Handoff page**: Detail page before external navigation (`/sites/*`, `/apis/*`).
