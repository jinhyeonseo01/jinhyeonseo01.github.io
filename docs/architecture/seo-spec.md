# SEO Spec

## Required metadata
- canonical per locale page
- hreflang set for all supported locales + `x-default`
- robots directive (`noindex` for redirect/non-content pages)
- Open Graph and Twitter summary metadata

## Structured data
- Dashboard pages:
  - `Person`
  - `WebSite`
  - `ItemList` for link catalog
- Detail pages:
  - `WebPage`
  - `WebSite` for destination context

## Sitemap policy
- `/sitemap.xml` generated by route endpoint
- Include localized dashboard and detail pages
- Exclude redirect-only routes from sitemap
