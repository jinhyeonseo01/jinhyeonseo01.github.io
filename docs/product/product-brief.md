# Product Brief

## Goal
Build a modern personal hub site that:
- introduces Jinhyeon Seo
- routes visitors to external channels (blog, YouTube, GitHub, projects)
- can scale to many projects and API doc links without IA refactoring

## Target audience
- Recruiters and collaborators
- Developers exploring projects and APIs
- Existing audience members coming from social/search

## Core outcomes
- Fast route from landing to destination links
- Strong SEO for localized pages
- Stable long-lived URL contract (`/go/{slug}`)
- Data-driven link expansion with minimal code edits
